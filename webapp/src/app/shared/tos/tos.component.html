La aplicación web para {{ subtitle }} es una herramienta para facilitar la información y decisiones propuganadas desde el Consejo Escolar del Centro, de forma que cualquier persona interesada tenga un acceso cómodo a esta información pública.

La aplicación no registra de facto ningún dato sobre el usuario, excepto nombre y correo electrónico sólo si el usuario envía sugerencias a través del formulario para dicho efecto. Esa información es totalmente opcional, puesto que el usuario puede enviar sus sugerencias sin necesidad de enviar sus datos personales. Esos datos se recavan opcionalmente sólo si el usuario desea que se le comunique una respuesta de forma directa.

En ese caso, tratamos la información que nos facilita con el fin de prestarles el servicio solicitado.
